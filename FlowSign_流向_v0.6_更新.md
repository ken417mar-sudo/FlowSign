# 流向 FlowSign — v0.6 更新文档

> v0.6 核心改动：去掉所有行动建议，只描述气势流向。
> 系统只做一件事：判断你当前所处的气势，描述它是什么。用户自己决定怎么应对。

---

## 一、哲学层改动

### v0.5 的定位（旧）

```
识别你的状态 + 告诉你怎么做
```

### v0.6 的定位（新）

```
读你当前所处的气势流向，描述它是什么
```

**区别：**

天气预报告诉你今天有风，不告诉你要不要出门。你知道有风，自己判断。流向做的是同样的事——读气势，不做决定。

系统能知道的是"你现在处在什么气势里"。系统不知道你今天有什么任务、面对什么人、有什么约束，所以不应该告诉你"应该做X"。

**去掉的内容：**

- 低耗路径
- 支撑路径
- 今日一件事
- 场景题（推进/关系/恢复）
- 所有"适合做X，不适合做Y"的表述

**留下的内容：**

- 3题输入（判断你的位置）
- 8种天气状态描述（这个气势是什么感觉）
- 气势走向（这个气势通常往哪里流）
- 一周天气报告（看自己的气势模式）
- 像不像反馈

---

## 二、8种天气状态描述（重写）

描述原则：
- 只描述这个气势的特征，不给建议
- 写能量在哪、注意力在哪、节奏是什么感觉
- 用户读完自己知道今天是什么状态，不需要系统说出来
- 语言像天气播报，不像心理咨询

---

### 晴天（开拓流）升·外·稳

```
今日天气：晴天 ☀️
```

**气势描述：**

能量充足，注意力自然朝向外部，状态清晰稳定。外部的人、事、机会都在你的感知范围内，你有足够的资源去接触它们。这种气势是向外扩展的，启动感强，方向感清楚。

**气势走向：**

持续高强度向外输出，容易流向有风。

---

### 有风（散射流）升·外·潮

```
今日天气：有风 🌬️
```

**气势描述：**

能量不低，但注意力被外部多个方向同时牵引。想法多，信息多，人和事都在涌来，感知是开放的，但方向不集中。节奏起伏，容易同时开很多线。

**气势走向：**

外部输入持续增加，容易流向风暴。收束注意力，可以流向阴天。

---

### 阴天（专注流）升·内·稳

```
今日天气：阴天 ☁️
```

**气势描述：**

能量充足，注意力自然向内收束。外部干扰对你的影响比平时小，更容易进入深度处理的状态。节奏稳定，不容易被打断，适合需要持续投入的事。

**气势走向：**

外部频繁介入，容易流向有风。保持向内，可以持续在阴天。

---

### 雷阵雨（燃烧流）升·内·潮

```
今日天气：雷阵雨 ⛈️
```

**气势描述：**

能量高，但内部有压力、急迫或情绪波动同时存在。动力和张力并存，节奏起伏明显。感知敏锐，但内部的噪音也更多，容易在高强度和过载之间切换。

**气势走向：**

持续高压，容易流向连阴天。内部收束，可以流向阴天。

---

### 小雨（周旋流）降·外·稳

```
今日天气：小雨 🌦️
```

**气势描述：**

能量偏低，但仍然需要面对外部事务。今天的资源是有限的，外部的人和事仍在运转。节奏稳定，不会有太大波动，但也没有多余的余力去扩展。

**气势走向：**

边界不清，容易流向风暴。保持稳定，可以慢慢流向多云。

---

### 风暴（漩涡流）降·外·潮

```
今日天气：风暴 🌪️
```

**气势描述：**

能量偏低，同时外部输入很多，节奏起伏。今天的资源不足以应对所有涌来的事，容易被外部的消息、任务、他人情绪带着走。边界感在这种气势下会变弱。

**气势走向：**

持续被外部牵引，容易流向连阴天。

---

### 多云（修整流）降·内·稳

```
今日天气：多云 🌥️
```

**气势描述：**

能量偏低，注意力向内，节奏稳定。这是一种安静的低能量状态，没有太多外部干扰，内部也相对平稳。气势是收缩的，但不是波动的。

**气势走向：**

这种气势通常是恢复性的，之后容易流向阴天。

---

### 连阴天（低潮流）降·内·潮

```
今日天气：连阴天 🌫️
```

**气势描述：**

能量低，注意力向内，内部有波动。这是资源最少的气势，外部和内部都在消耗。感知是收缩的，决策和社交在这种气势下都需要更多能量。

**气势走向：**

如果连续出现，系统会提示趋势变化。

### 文案原则（更新）

使用：
- 能量在……
- 注意力……

- 适合做……
- 不适合做……
- 建议你……
