# Changelog

## v0.5 — 2026-05-13

- 整体方案切换为「天气映射」：天气名作为用户可见表达，流向名作为系统内部分类
- 新增 8 种内在天气：晴天、有风、阴天、雷阵雨、小雨、风暴、多云、连阴天
- 结果页天气化：首屏展示今日天气，同时保留对应流向和三维解释
- 分享卡天气化：从“我今天是燃烧流”改为“我今天雷阵雨”
- 一周报告天气化：从“流向分布”升级为“本周天气”
- 小红书验证加入命名 A/B 测试：流向名 vs 天气名
- README 更新为 v0.5 天气映射方向

## v0.4 — 2026-05-12

- 项目名统一为「流向 FlowSign」
- 固定三题改为题库轮换 + 自适应追问
- A/B 选项中性化，减少好坏暗示
- 增加“不太像”后的即时校正与个人权重记录
- 本流 × 日流改成维度差异规则
- 补齐趋势触发、分享卡映射、断用返回、重测本流机制
