# Changelog

## v0.8 — 2026-05-13

逻辑校验修复，六个问题全部修正：

1. **新增输入系统章节**：固定信息层/时间环境层/今日校准层，每层有理论依据和作用说明
2. **出生日期改为出生季节**：简化为春/夏/秋/冬，映射到季节节律敏感度
3. **去掉月相**：科学证据薄弱，不纳入核心算法
4. **3题固定对应关系**：证据题1→能量维，证据题2→方向维，投射题→节奏维
5. **节奏维补理论**：加入超日节律（ultradian rhythm）的科学依据
6. **本流重新定义**：从"基础频率"改为"近期节律模式"，明确会随时间更新
7. **气势走向去条件句**：只描述自然倾向，不加"如果你做了X"
8. **结果页更新为v0.6版本**：去掉低耗路径/支撑路径/今日一件事
9. **今日场景题标记移除**：v0.6已决定去掉，文档同步

## v0.7 — 2026-05-13

- 世界观全面重写：以西方哲学（赫拉克利特、勒温场论）+ 现代科学（时间生物学、心流理论）为理论基础
- 核心概念从"状态识别"升级为"节律共振"：你的气势是个人节律与时间场节律相遇的结果
- 去掉中国传统气论的直接表述，保留底层逻辑，用科学语言重新包装
- 产品定位更新：从"状态镜子+行动建议"改为"当下气势的读数"
- 价值观语言更新：顺势→读势，好坏流向→好坏气势

## v0.6 — 2026-05-13

- 产品哲学根本改动：去掉所有行动建议，系统只判断气势，用户自己决定怎么应对
- 8种天气状态描述全部重写：只描述气势特征（能量/注意力/节奏），不给建议
- 结果页简化：天气名 + 气势描述 + 气势走向 + 像不像反馈
- 去掉：低耗路径、支撑路径、今日一件事、场景题（推进/关系/恢复）
- 定位语言更新：从"识别状态+行动建议"改为"读气势流向"

## v0.5 — 2026-05-13

- 整体方案切换为「天气映射」：天气名作为用户可见表达，流向名作为系统内部分类
- 新增 8 种内在天气：晴天、有风、阴天、雷阵雨、小雨、风暴、多云、连阴天
- 结果页天气化：首屏展示今日天气，同时保留对应流向和三维解释
- 分享卡天气化：从“我今天是燃烧流”改为“我今天雷阵雨”
- 一周报告天气化：从“流向分布”升级为“本周天气”
- 小红书验证加入命名 A/B 测试：流向名 vs 天气名
- README 更新为 v0.5 天气映射方向
