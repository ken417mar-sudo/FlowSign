# 流向 FlowSign

> 你的内在天气预报 / 读你此刻的节律气势

---

## 一句话

**流向 FlowSign，读你当前所处的气势流向。**

你有自己的节律底色，当前时刻有它的节律。你此刻的气势，是这两个节律相遇的状态。流向读的就是这个相遇。

```text
天气 = 用户可见表达
流向 = 系统内部分类
```

> 每天一张卡，看见你今天的气势在哪里。

---

## 核心定位

**当下气势的读数。**

流向不预测命运，不固定性格，不告诉你应该做什么。它只做一件事：

读你此刻的节律状态，告诉你现在处在什么气势里。

就像指南针读的是磁场，不是你应该去哪里。

## 系统结构

### 三维状态模型（MVP）

```text
身体资源 × 注意力朝向 × 信息节奏 = 8 种基础内在天气
```

| 维度 | 两极 | 描述 |
|------|------|------|
| 能量维 | 升 / 降 | 今天的身心资源是上升还是下降 |
| 方向维 | 外 / 内 | 注意力朝向外部世界还是内部世界 |
| 节奏维 | 稳 / 潮 | 倾向线性稳定还是波动并行 |

### 8 种内在天气

| 组合 | 天气名 | 对应流向 | 核心状态 |
|------|------|------|------|
| 升·外·稳 | 晴天 | 开拓流 | 有能量，向外推进 |
| 升·外·潮 | 有风 | 散射流 | 想法多，外部牵引多 |
| 升·内·稳 | 阴天 | 专注流 | 有能量，适合深度处理 |
| 升·内·潮 | 雷阵雨 | 燃烧流 | 有动力，但夹杂压力和内耗 |
| 降·外·稳 | 小雨 | 周旋流 | 能量不高，但仍需处理外部事务 |
| 降·外·潮 | 风暴 | 漩涡流 | 被外界卷动，边界感变弱 |
| 降·内·稳 | 多云 | 修整流 | 低能但稳定，适合恢复整理 |
| 降·内·潮 | 连阴天 | 低潮流 | 能量低、情绪波动，需要保护自己 |

---

## 三层时间系统

```text
本流 Base Flow      — 长期默认状态节律
日流 Daily Flow     — 今天的主基调（MVP 核心）
时刻流 Moment Flow  — 一天内的局部波动（后续扩展）
```

## MVP 范围

**第一阶段必须做：**
- 每日核心输入与自适应追问（默认3题，低置信度追加1-2题）
- 8 种内在天气结果页（低耗路径 / 支撑路径 / 今日一件事）
- 分享卡
- 像不像反馈与校正流程
- 连续低潮趋势提示
- 小红书内容验证（14天计划）

**第一阶段暂缓：**
- 出生日期强绑定
- 首次本流测试
- 完整16流向
- 重 App 开发
