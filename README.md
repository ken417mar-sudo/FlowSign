# 流向 FlowSign

> 每日状态节律识别系统 / Daily State Flow Navigation System

---

## 一句话

**流向 FlowSign，是一套每日状态节律识别系统。**

它不预测命运，也不固定人格，而是通过每日直觉选择，帮助用户识别今天的状态主基调，理解自己当下的能量、注意力和节奏，并获得下一步行动建议。

> 每天一张卡，识别你现在处在哪条状态流里，并告诉你怎么走更顺。

---

## 核心定位

- 不是命理预测（不说"你今天会遇到贵人"）
- 不是性格测试（不给你贴永久标签）
- 是**当下状态的镜子** + **下一步行动建议**

类比：天气预报不决定你的命运，但它帮你决定今天穿什么、带不带伞。流向做的是你内在状态的天气预报。

## 系统结构

### 三维状态模型（MVP）

```
身体资源 × 注意力朝向 × 信息节奏 = 8 种基础流向
```

| 维度 | 两极 | 描述 |
|------|------|------|
| 能量维 | 升 / 降 | 今天的身心资源是上升还是下降 |
| 方向维 | 外 / 内 | 注意力朝向外部世界还是内部世界 |
| 节奏维 | 稳 / 潮 | 倾向线性稳定还是波动并行 |

### 8 种基础流向

| 组合 | 名称 | 核心状态 |
|------|------|------|
| 升·外·稳 | 开拓流 | 有能量，向外推进 |
| 升·外·潮 | 散射流 | 想法多，外部牵引多 |
| 升·内·稳 | 专注流 | 有能量，适合深度处理 |
| 升·内·潮 | 燃烧流 | 有动力，但夹杂压力和内耗 |
| 降·外·稳 | 周旋流 | 能量不高，但仍需处理外部事务 |
| 降·外·潮 | 漩涡流 | 被外界卷动，边界感变弱 |
| 降·内·稳 | 修整流 | 低能但稳定，适合恢复整理 |
| 降·内·潮 | 低潮流 | 能量低、情绪波动，需要保护自己 |

---

## 三层时间系统

```
本流 Base Flow    — 长期默认状态节律
日流 Daily Flow   — 今天的主基调（MVP 核心）
时刻流 Moment Flow — 一天内的局部波动（后续扩展）
```

## MVP 范围

**第一阶段必须做：**
- 每日核心输入与自适应追问（默认3题，低置信度追加1-2题）
- 8 种流向结果页（低耗路径 / 支撑路径 / 今日一件事）
- 分享卡
- 像不像反馈与校正流程
- 连续低潮趋势提示
- 小红书内容验证（14天计划）

**第一阶段暂缓：**
- 出生日期强绑定
- 首次本流测试
- 完整16流向
- 重 App 开发
