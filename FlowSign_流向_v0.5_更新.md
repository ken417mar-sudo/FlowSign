# 流向 FlowSign — v0.5 更新文档

> v0.5 不扩世界观，只解决三件事：
> 1. 日流题目重构（投射 + 今日证据，区分日流和本流）
> 2. 行动建议个性化（流向 × 场景）
> 3. 一周流向报告（第8天留存钩子）

---

## 一、日流题目重构

### 问题回顾

v0.4 的投射题（想象一朵云、想象一团火）测的是长期性格倾向，而不是今天的状态。内向用户每天都会选"静止"，外向用户每天都会选"流动"，导致日流结果高度重复，等于在测本流。

### v0.5 输入结构

```
1 道意象投射题（测倾向，保留仪式感）
+ 2 道今日证据题（测今天，锚定当下）
= 3 题，约 15 秒
```

投射题保留 1 道，维持仪式感入口。证据题新增 2 道，用今天已经发生的行为锚定状态，减少长期性格的污染。

---

### 意象投射题题库（每维度 4 题，每天随机抽 1 题）

每天只出 1 道投射题，从三个维度里随机选一个维度出题。

**能量维**

```
如果今天的你是一阵潮水，它更像——
A. 正在涨上来，把东西推向岸边
B. 正在退下去，露出下面的轮廓
```

```
如果今天的你是一团火，它更像——
A. 火苗往上走，想把东西点起来
B. 火光往里收，适合慢慢保温
```

```
如果今天是一段路，你更像走在——
A. 开始有坡度的上行路
B. 需要保存体力的缓行路
```

```
如果今天的你是一棵树，它更像——
A. 枝叶在往外伸，想接触更多阳光
B. 根系在往下扎，需要安静积蓄
```

**方向维**

```
如果今天的注意力是一束光，它更容易照向——
A. 外面的声音、人和变化
B. 自己的想法、感受和判断
```

```
如果今天的你是一间房，门口的状态更像——
A. 门半开着，外面的东西容易进来
B. 门虚掩着，里面的空间更清楚
```

```
如果今天有很多信息经过你身边，你更像——
A. 会自然伸手接住一些
B. 会先让它们从旁边经过
```

```
如果今天的你是一个容器，它更像——
A. 开口朝外，准备接收和输出
B. 开口朝内，需要先整理里面
```

**节奏维**

```
如果今天的脑子是一条河，它更像——
A. 河道清楚，水往一个方向走
B. 支流很多，几股水同时在动
```

```
如果今天的状态是一张桌面，它更像——
A. 东西不多，能一件件拿起来
B. 东西不少，但每样都像有关系
```

```
如果今天的想法是一片天空，它更像——
A. 云层分明，能看清远处
B. 云在流动，形状一直变化
```

```
如果今天的节奏是一首曲子，它更像——
A. 节拍稳定，能跟上
B. 旋律起伏，情绪带着走
```

### 今日证据题题库（固定 2 道，每天必出）

证据题不轮换，每天固定出，因为它们测的是今天的客观行为，不存在"看穿规律"的问题。

**证据题 1：启动感（测能量维）**

```
从今天醒来到现在，你完成事情的启动感更像——

A. 比较容易开始，已经推进了一些事
B. 能开始，但需要一点推力
C. 很难启动，更多是在撑着
```

映射：A → 升，B → 中性，C → 降

**证据题 2：主被动（测方向维）**

```
从今天醒来到现在，你更多是——

A. 主动联系、发起或推进了一些事
B. 主要在回应别人或处理找上来的事
C. 基本没有对外互动，主要在自己这里
```

映射：A → 外，B → 中性，C → 内

---

### 置信度判断更新

| 信号 | 说明 |
|------|------|
| 投射题 + 证据题一致 | 高置信度，直接出结果 |
| 投射题和证据题矛盾 | 低置信度，追加 1 道节奏校验题 |
| 证据题选 B（中性） | 中置信度，结果页加"今天状态摇摆"提示 |
| 历史记录连续 3 天相同 | 触发"你最近状态很稳定"提示，而非重复出相同结果页 |

## 二、行动建议个性化

### 问题回顾

v0.4 的行动建议对所有人每天都成立（"选一个目标推进"、"减少输入"），个性化没有落地。

### v0.5 方案：流向 × 场景

在结果页之前，加一道场景题。场景题只有 3 个选项，不参与流向判断，只影响行动建议的语言和方向。

**场景题**

```
今天你最需要面对的是——

A. 推进事情（工作、任务、目标）
B. 处理关系（沟通、协作、社交）
C. 恢复自己（休息、情绪、身体）
```

---

### 行动建议矩阵（8 流向 × 3 场景 = 24 条）

#### 开拓流（升·外·稳）

| 场景 | 建议 |
|------|------|
| 推进事情 | 选一个明确目标，今天可以推得比平时更远。主动发起，不要等别人来找你。 |
| 处理关系 | 今天适合主动沟通、推进合作或做明确表态。对方会感受到你的清晰。 |
| 恢复自己 | 今天状态好，但恢复不等于浪费。可以做一件平时没时间做的"想做的事"。 |

#### 散射流（升·外·潮）

| 场景 | 建议 |
|------|------|
| 推进事情 | 今天想法很多，先只选一个推进。其他想法记下来，不要同时开。 |
| 处理关系 | 今天容易被对话带着走。沟通前先想好自己的核心诉求，不要被带偏。 |
| 恢复自己 | 减少信息输入。今天不适合刷内容，适合散步、听音乐、做一件手上的事。 |

#### 专注流（升·内·稳）

| 场景 | 建议 |
|------|------|
| 推进事情 | 今天适合深度工作。留一段完整时间，关掉干扰，把一件重要的事做深。 |
| 处理关系 | 今天更适合书面沟通而不是即时对话。先想清楚再表达，效果会更好。 |
| 恢复自己 | 今天的恢复方式是独处和整理。不需要社交，给自己一段安静时间。 |

#### 燃烧流（升·内·潮）

| 场景 | 建议 |
|------|------|
| 推进事情 | 今天有动力但容易过载。只选一个任务，拆成 30 分钟以内的小段，设停止点。 |
| 处理关系 | 今天情绪敏感度高，沟通前先写草稿，不要即时回复重要消息。 |
| 恢复自己 | 今天需要给能量一个出口，而不是压着。短时间高强度的事（运动、写作）比休息更有效。 |

#### 周旋流（降·外·稳）

| 场景 | 建议 |
|------|------|
| 推进事情 | 今天不用冲，只处理必须完成的事。把目标压缩到最小完成线。 |
| 处理关系 | 今天适合低成本周旋。提前准备要点，少解释，多确认，不主动扩大话题。 |
| 恢复自己 | 今天能量不高，恢复的方式是减少决策。把今天要做的事提前列好，不要临时想。 |

#### 漩涡流（降·外·潮）

| 场景 | 建议 |
|------|------|
| 推进事情 | 先列出"必须今天完成"和"可以延后"两类，只处理第一类。不要全部接住。 |
| 处理关系 | 今天容易被别人的情绪带走。设一个边界：今天只回应最必要的沟通。 |
| 恢复自己 | 今天最重要的事是降低输入。关掉不必要的通知，给自己一段不被打扰的时间。 |

#### 修整流（降·内·稳）

| 场景 | 建议 |
|------|------|
| 推进事情 | 今天适合处理熟悉的、有收尾感的任务。不要开新坑，把之前没完成的事收一收。 |
| 处理关系 | 今天不适合主动发起重要沟通。如果必须，先把要说的写下来，再决定要不要发。 |
| 恢复自己 | 今天的恢复方式是整理。整理环境、整理清单、整理思路，都比强迫休息更有效。 |

#### 低潮流（降·内·潮）

| 场景 | 建议 |
|------|------|
| 推进事情 | 今天只保留最低完成线。允许自己做得不完美，完成比完美重要。 |
| 处理关系 | 今天不适合处理复杂关系。非必要的沟通可以延后，不要在低能量时做重要表态。 |
| 恢复自己 | 今天最重要的事是不要继续消耗自己。做一件 5 分钟内能完成的小事，然后停下来。 |

## 三、一周流向报告

### 定位

一周报告是第 8 天的留存钩子。它不是数据汇总，而是让用户看见自己的**状态模式**——"原来我每次进入漩涡流之前，都先经过散射流"。

### 触发条件

```
累计完成 7 次日流记录（不要求连续）
```

触发后，每完成 7 次新记录，自动生成新一期报告。

### 报告结构

```
你的近期流向
─────────────────────────────
过去 7 次记录分布
[开拓流 ██░░░░░ 2次]
[专注流 ████░░░ 3次]
[燃烧流 ██░░░░░ 2次]
─────────────────────────────
你最常回到的状态
专注流 · 3次

高消耗状态出现
燃烧流 2次 / 漩涡流 0次
─────────────────────────────
你的状态模式
你最近 7 次里有 2 次燃烧流，
通常出现在专注流之后。
这可能说明你在深度工作后
容易进入压力叠加的状态。
─────────────────────────────
下一步建议
专注流之后，主动设一个停止点，
不要让深度工作直接连接高压任务。
─────────────────────────────
```

### 模式识别规则（MVP 版）

不需要复杂算法，先用规则引擎实现：

| 模式 | 触发条件 | 解释文案方向 |
|------|------|------|
| 稳定型 | 7次中同一流向 ≥ 4次 | 你最近状态很稳定，一直在X附近 |
| 高耗型 | 燃烧流+漩涡流+低潮流合计 ≥ 4次 | 你最近高消耗状态较多，需要注意恢复 |
| 转化型 | 出现明显的流向转化链路 | 你最近从X流向了Y，这个转化通常意味着… |
| 波动型 | 7次中出现 5种以上不同流向 | 你最近状态变化比较多，可能外部环境变化较大 |
| 回升型 | 后3次比前4次能量维更高 | 你最近状态在回升，可以开始推进之前搁置的事 |
| 下行型 | 后3次比前4次能量维更低 | 你最近状态在下行，建议减少高消耗任务 |

### 报告文案原则

- 描述模式，不做评判（"你最近经常进入燃烧流"，不是"你最近状态不好"）
- 给出一个具体的下一步，不是泛泛建议
- 长度控制在 150 字以内，用户 30 秒能读完

## 四、结果页更新

### v0.5 完整结果页结构

```
今日流向：[名称] [符号]
[一句话]
─────────────────────────────
今日状态
[2-3句状态描述]
─────────────────────────────
今天你最需要面对的是——
○ 推进事情  ○ 处理关系  ○ 恢复自己
─────────────────────────────
[场景对应的行动建议，2-3句]
─────────────────────────────
今天只做一件事
[1条具体行动]
─────────────────────────────
风险提醒
[1条需要注意的倾向]
─────────────────────────────
像你吗？
[很像]  [有点像]  [不太像]
─────────────────────────────
[分享这张卡]
```

场景选择放在行动建议之前，用户选完场景，建议立刻更新，有即时反馈感。
